# COBRA Framework

> Task dispatch, worker registration, and annotation-based discovery for interpreter and analysis systems.

Kotlin library providing `IWorker`, `IDispatcher`, `AbcWorkshop`, and `WorkLicense` abstractions. Workers process tasks and return results synchronously. Workshops group workers and expose them through annotation-based licensing. Dispatchers route tasks to registered workers by `ITask.ID`.

## Instructions for Agents

- Subclass `AbcWorkshop<W>` to group related workers. Annotate worker properties with a `@WorkLicense`-annotated annotation.
- Call `licensedWorkers()` on a workshop instance to get a `Map<ITask.ID, W>` of all annotated workers.
- Implement `IDispatcher<W>` to manage worker registration and lookup by `ITask.ID`.
- Implement `IWorker<T, R>` with `fun work(task: T): R`. Workers are synchronous вҖ” concurrency managed by the caller.
- Use `WorkLicense.getTaskID(annotation)` to extract `ITask.ID` from a `@WorkLicense`-annotated annotation instance.
- `@WorkLicense` is a meta-annotation. Apply it to custom annotation classes, not to properties directly.
- Result type `R` is caller-defined: `Boolean`, `Unit`, or domain-specific types.

# Core

> Task dispatch, worker execution, and annotation-based licensing.

## Quick Start

```kotlin
@Target(AnnotationTarget.PROPERTY)
@Retention(AnnotationRetention.RUNTIME)
@WorkLicense
annotation class MyLicense(val name: String)

class MyWorkshop : AbcWorkshop<IWorker<MyTask, Boolean>>() {
    @MyLicense("greet")
    val greeter = IWorker<MyTask, Boolean> { task -> true }
}

val workers = MyWorkshop().licensedWorkers()
```

## API

**`ITask`** вҖ” Unit of work with `uid: ITask.ID`.

**`ITask.ID(license: String, props: Set<String>)`** вҖ” Dispatch key. Constructed via `WorkLicense.getTaskID()`.

**`IWorker<T : ITask, R>`** вҖ” Processes a task, returns a result. `fun interface`.

**`fun work(task: T): R`** вҖ” Synchronous. Result type `R` determined by caller.

**`IDispatcher<Worker : IWorker<*, *>>`** вҖ” Routes tasks to workers.

**`fun dispatch(forTask: ITask.ID): Worker?`** вҖ” Returns registered worker or null.

**`fun register(forTask: ITask.ID, toWorker: Worker)`** вҖ” Binds worker to task ID.

**`AbcWorkshop<W : IWorker<*, *>>`** вҖ” Groups workers; discovers via reflection.

**`fun licensedWorkers(): Map<ITask.ID, W>`** вҖ” Scans `@WorkLicense`-annotated properties.

**`@WorkLicense`** вҖ” Meta-annotation for custom license annotation classes.

**`WorkLicense.getTaskID(annotation: Annotation): ITask.ID`** вҖ” Extracts ID from annotation instance.

**`WorkLicense.getTaskID(cls: Class<*>, vararg props: String): ITask.ID`** вҖ” Constructs ID from class + props.

**`WorkLicense.isTaskID(taskID: ITask.ID, forLicense: Class<*>): Boolean`** вҖ” Checks ID matches license class.

## Configuration

No configuration required. Behavior driven by annotations and registration calls.

## Gotchas

- `@WorkLicense` targets `ANNOTATION_CLASS`, not `PROPERTY`. Applying directly to a worker property has no effect.
- `licensedWorkers()` uses `kotlin-reflect`. The `kotlin-reflect` artifact must be on the classpath.
- `licensedWorkers()` reads `declaredMemberProperties` вҖ” inherited properties not discovered.
- `ITask.ID.license` stores `Class.simpleName`. Same simple name in different packages вҶ’ identical license.
- `IWorker.work` is synchronous. Callers manage concurrency (coroutines, thread pools).
